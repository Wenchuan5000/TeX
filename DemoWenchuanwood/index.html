Differentiable Mappings

Definition

Let f:Rm→Rn, and let p∈Rm. f is differentiable at p iff there exists a linear map ϕ:Rm→Rn, such that for any t∈Rm,

limt→0Rnf(p+t)−f(p)−ϕ(t)∥t∥Rm=0Rn.

Recall that for any f:Rm→Rn,

limx→x0r(x)∈Rn

iff there exists some α:Rm→Rn with α(x)→0Rn as x→x0, such that there exists an open subset U⊆Rm with x∈U, such that for any x∈U,

f(x)=limx→x0f(x)+α(x).

Thus, in the notes below, the assertion there exists some α:Rm→Rn with α(t)→0Rn as t→0Rm, such that there exists an open subset U⊆Rm with p∈U, such that for any t∈Rn with p+t∈U,

f(p+t)−f(p)−ϕ(t)∥t∥Rm=0Rn

is straightly used as equivalent as ref when it is refered sometime.

Lemma

The linear map ϕ in ref is unique.

Proof. As f is differentiable at p, there exists an α:Rm→Rn with α(t)→0Rn, such that there exists an open subset U⊆Rm with p∈U, such that for any t∈Rn with p+t∈U,

f(p+t)−f(p)−ϕ(t)∥t∥Rm=α(t).

Assume ref holds for another linear mapping λ also, then, similarly, there exists a β:Rm→Rn with β(t)=0Rn, such that there exists an open subset U′⊆Rm with p∈U′, such that for any t∈Rn with p+t∈U′,

f(p+t)−f(p)−λ(t)∥t∥Rm=β(t).

Let γ=ϕ−λ. As ϕ and −λ are both linear, then

 γ(t)∥t∥Rm=α(t)−β(t)⟺limt→0Rmγ(t^)=limt→0Rm(α(t)−β(t))⟺ γ(t^)=0Rn.

As t is arbitrarily picked from U∩U′, and U∩U′ is open in Rm as U and U′ are open, the set

{t^:t∈U∩U′−p}

gives all possible directions in Rm. And as γ(st^)=0Rn, for any s∈R, γ(t)=0Rn constantly. Thus, ϕ=λ.

Lemma

With the condition in ref, if f is differentiable at p, then f is continuous at p.

Proof. As f is differentiable at p, there exists an α:Rm→Rn with α(t)→0Rn as t→0Rm, such that such that there exists an open subset U′⊆Rm with p∈U′, such that for any t∈Rn with p+t∈U′,

f(p+t)−f(p)−ϕ(t)∥t∥Rm=α(t).

Then, we have

 limt→0Rm[f(p+t)−ϕ(t)]=limt→0Rm[∥t∥Rmα(t)+f(p)]⟺ limt→0Rmf(p+t)=f(p).

This, implies f is continuous at p.

Directional Derivatives

Definition

Let f:Rm→Rn, let u∈Rm∖{0Rm}, and let p∈Rm. The directional derivative of f along u at p is defined as

∇uf(p):=limt→0f(p+tu)−f(p)t,

if the limit exists in Rn.

Lemma

With the condition in ref, if ∇uf(p) exists, then there exits an open subset U⊆Rm with p∈U such that f is relative continuous on the line defined by p+tu for some t∈R.

Proof. Let U be an open subset of Rm, and let g:R→Rm be defined as

g(t):=p+tu.

Then f is relative continuous on the line defined by p+tu for some t∈R iff f↾g[R] is continuous on U∩g[R].

Let h=f∘g, then

∇uf(p)=limt→0h(t)−h(0)t=v∈Rn.

Then, there exists an α:R→Rn with α(t)→0Rn as t→0, such that there exists an open subset I⊆R with 0∈I, such that for any t∈I,

h(t)=tv+tα(t)+h(0).

Then we have

 limt→0h(t)=limt→0[tv+tα(t)+h(0)]⟺ limt→0h(t)=h(0).

Thus, h is continuous at 0.

It is easy to find that g is bijective, thus g−1 is also bijective. Thus, g∘g−1 is an identity mapping on g[R]⊆Rm. As composition of mappings is associative, we have

h=f∘g⟺h∘g−1=f∘g∘g−1⟺h∘g−1=f∘(g∘g−1)⟺h∘g−1=f↾g[R].

It is also easy to find that g−1 is continuous everywhere, thus, as h is continuous at 0, f↾g[R] is continuous on U∩g[R]. Thus, f is relative continuous on the line defined by p+tu for some t∈R.
