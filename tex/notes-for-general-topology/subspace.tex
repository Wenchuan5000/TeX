	A collection $\mathcal T_S \subseteq \mathcal P(S)$ is a \textit{subspace topology} on $S$ if and only if for any $U_S \subseteq S$, there is a $U \in \mathcal T$, such that $S \cap U = U_S$; i.e.,
	$$

\end{definition}
